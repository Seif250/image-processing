# لقطة شاشة للتطبيق

قم بإضافة لقطة شاشة للتطبيق في هذا الملف قبل رفعه على GitHub.

يمكنك القيام بذلك من خلال:
1. تشغيل التطبيق
2. فتح صورة وتطبيق بعض المعالجات عليها
3. التقاط لقطة شاشة للتطبيق (يمكنك استخدام مفتاح PrtScr أو أداة Snipping Tool)
4. حفظ لقطة الشاشة بصيغة PNG أو JPG
5. وضعها في مجلد المشروع الرئيسي باسم "screenshot.png" أو "screenshot.jpg"

سيؤدي ذلك إلى تحسين عرض مشروعك على GitHub وجعله أكثر جاذبية للمستخدمين.

بعد إضافة لقطة الشاشة، قم بإضافة الصورة إلى ملف README.md بهذه الطريقة:

```
## لقطة شاشة للتطبيق

![لقطة شاشة للتطبيق](screenshot.png)
```

ملاحظة: يمكنك حذف هذا الملف النصي بعد إضافة لقطة الشاشة.
